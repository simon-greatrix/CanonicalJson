# Release Notes — 3.1

**Released:** 2025-04-15

## Overview

Version 3.1 is a breaking change release that renames the library's package from `io.setl.json` to `com.pippsford.json`, reflecting a change in the project's group ID and Maven coordinates. All consumers must update their import statements.

## Breaking Changes

- **Package rename:** All classes have moved from the `io.setl.json.*` package hierarchy to `com.pippsford.json.*`. Update all import statements accordingly.
- **Group ID change:** The Maven/Gradle group ID has changed from `io.setl` to `com.pippsford`. Update dependency declarations.
- **Service provider files** updated to reference the new package.
- Spring auto-configuration entries in `META-INF/spring.factories` updated to the new package.

## Changes

- Significantly expanded test coverage across I/O, builder, exception, pointer, and parser classes. Many new test classes added:
  - `AppendableOutputTest`, `DelegatingGeneratorTest`, `GeneratorFactoryTest`, `InputTest`, `NoOpFormatterTest`, `PrettyFormatterTest`, `WriterFactoryTest`
  - Exception tests: `IncorrectDigestExceptionTest`, `IncorrectValueExceptionTest`, `InvalidPatchExceptionTest`
- `PrettyEmptyStructure` added for improved pretty-printing of empty JSON structures.
- `PrettyFormatter` and `PrettyOutput` substantially reworked.
- `TrustedGenerator` improved.
- `CJReader` refactored for correctness and coverage.
- `AppendableOutput` expanded with new capabilities.
- `GeneratorFactory` reworked.
- `Test` patch operation refactored into `TestDigest`, `TestResult`, and `TestValue` sub-classes for clearer separation of concerns.
- `CJStructure` marker interface added.
- `upload.sh` script added for publishing.
- Updated Maven Central publishing configuration.
- Code quality and checkstyle improvements throughout.
