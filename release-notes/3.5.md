# Release Notes — 3.5

**Released:** 2026-01-18

## Overview

Version 3.5 adds a full Jackson 3 integration module alongside the existing Jackson 2 support. It also includes dependency upgrades and miscellaneous fixes.

## New Features

- **Jackson 3 support:** A new `com.pippsford.json.jackson3` package provides a complete parallel integration for [Jackson 3.x](https://github.com/FasterXML/jackson):
  - `CanonicalFactory` — `JsonFactory` implementation producing canonical JSON output.
  - `CanonicalGenerator` — `JsonGenerator` that writes canonical-format JSON.
  - `CanonicalJsonModule` — Jackson `Module` registering all serialisers and deserialisers.
  - `CanonicalSerializer` / `CJBaseSerializer` — serialisers for `JsonValue` types.
  - `Convert` — bidirectional conversion between Jackson `JsonNode` and `jakarta.json.JsonValue`.
  - `JacksonGenerator` / `JacksonReader` — generator and reader bridging Jackson 3 and the canonical API.
  - Serialisers and deserialisers for `JsonArray`, `JsonObject`, `JsonStructure`, `JsonValue`, and `JsonPatch`.
  - Registered via `META-INF/services/tools.jackson.databind.JacksonModule`.
  - Full test suite covering all new classes.

## Other Changes

- `Jackson2` `Convert` class updated to align with the Jackson 3 API surface.
- `JsonModule` (Jackson 2) updated for compatibility.
- Dependency upgrades and `build.gradle` updates.
- `settings.gradle` updated.
- `IncorrectTypeException` and `MissingItemException` minor fixes.
- `EmptyPointer` and `CJBase` minor corrections.
- `CanonicalFactoryTest` and `JacksonReaderTest` (Jackson 2) extended.
- `PatchTest` extended with additional cases.
- `Pojo` test class relocated to `objects` sub-package for consistency.
