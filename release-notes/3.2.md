# Release Notes — 3.2

**Released:** 2025-04-22

## Overview

Version 3.2 is a minor release focused on API refinement. Factory and provider methods that previously returned generic `jakarta.json` types now return the library's own `Canonical` subtypes, giving callers richer access to canonical-specific behaviour without casting.

## Changes

- **`Canonical` return types:** Methods across `Canonical`, `CanonicalCreator`, `CanonicalJsonProvider`, `BuilderFactory`, `WriterFactory`, `ParserFactory`, and related classes now return `Canonical`-typed objects instead of the plain `jakarta.json` interfaces.
- **`CJParser` added:** A new `CJParser` class in the parser package provides a canonical-aware parser implementation.
- `ArrayWalker` and `ObjectWalker` updated to use the new return types.
- `StructureParser` enhanced to work with the new parser hierarchy.
- `WalkingParser` refactored.
- `PatchBuilder` and `PatchFactory` updated for consistency.
- Minor fix to the `upload.sh` publishing script.
