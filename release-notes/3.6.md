# Release Notes — 3.6

**Released:** 2026-01-18

## Overview

Version 3.6 removes the Spring integration classes that were added in earlier releases and fixes streaming capabilities in the Jackson 3 `CanonicalGenerator`. Users who relied on the Spring auto-configuration must now configure the library manually.

## Breaking Changes

- **Spring integration removed:** The following classes have been deleted and are no longer available:
  - `com.pippsford.json.spring.ConfigureCanonicalJson`
  - `com.pippsford.json.spring.ConfigureCanonicalWebMvc`
  - `com.pippsford.json.spring.CanonicalWebMvcConfigurer`
  - The `META-INF/spring.factories` auto-configuration entry has been removed.

## Bug Fixes

- **`CanonicalGenerator` (Jackson 3):** Fixed streaming output capabilities so that the generator correctly handles incremental / streaming write scenarios.

## Migration Guide

If you were relying on Spring auto-configuration, add explicit configuration in your Spring application:

```java
@Configuration
public class JsonConfig implements WebMvcConfigurer {
    @Override
    public void configureMessageConverters(List<HttpMessageConverter<?>> converters) {
        // Configure canonical JSON converters manually
    }
}
```
